# НаРаботеLIVE ANDROID WEB

Мобильная серверная версия приложения НаРаботеLIVE для Android и браузера.

## Deploy to Render (бесплатно)

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/klon312-OWNED/NaRaboteLIVE-AndroidWeb)

## Запуск

```bash
npm install
npm start              # сервер на порту 3000
npm run start:tunnel   # сервер + туннель для доступа из интернета
```

## Использование

1. Запустите сервер на ПК или задеплойте на Render
2. Откройте URL сервера в браузере телефона
3. Или установите APK-приложение и введите адрес сервера
