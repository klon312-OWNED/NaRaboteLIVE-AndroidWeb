# НаРаботеLIVE ANDROID WEB

Мобильная серверная версия приложения НаРаботеLIVE для Android и браузера.

## Запуск

```bash
npm install
npm start              # сервер на порту 3000
npm run start:tunnel   # сервер + туннель для доступа из интернета
```

## Использование

1. Запустите сервер на ПК
2. Откройте `http://IP_ПК:3000` в браузере телефона
3. Или установите APK-приложение и введите адрес сервера
